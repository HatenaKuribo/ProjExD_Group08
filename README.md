# シューティングゲーム

## 実行環境の必要条件

python >= 3.10

* pygame >= 2.1

## ゲームの概要

* 敵を倒してスコアを増やしていく、スコアアタック型シューティングゲーム

## ゲームの遊び方

* 始めるとキャラ選択画面に移行、お好きなキャラを選ぶ
* 選ぶとゲームが始まる、Zキーで射撃、矢印キーで上下左右に移動
* Shiftを押しながらだと低速で移動できる(精密な動きに使えます)
* 敵を倒すとスコアがもらえる
* 一度でも敵、もしくは敵の弾に当たるとゲームオーバー
* 一定スコアを得るとボスが出現
* ボスを倒すとまた次のボス出現までスコアを稼ぐ、これの繰り返し

## ゲームの実装

### 共通基本機能

* タイトル画面
* キャラ選択画面
* デフォルトの2キャラ
* ゲーム画面
* 自キャラの挙動
* 敵キャラの挙動
* ゲームオーバー画面

### 分担追加機能

* 全員がそれぞれキャラクターを作成、追加する
* 2way-1wayで射撃を切り替えられるキャラ:担当(自分)
